FreeEMS Tools

This page exists to share information on how to obtain various tools for DIY EFI projects. In particular, it exists to share the tools required to build FreeeEMS with the world.

  • For Debian and Ubuntu there is a repository available. Add the following to your sources.list file (found in /etc/apt/) deb debs.diyefi.org css-debs binary then run apt-get update then run apt-get install binutils-mc9xgate binutils-mc9s12x gcc-m68hc1x newlib-m68hc1x. Alternatively you may download the debian archives from the following location and install them manually with dpkg -i *.deb : Debian Archives
  • For Windows (XP/Vista/7) you can get the installer from right here: FreeEMS-ToolChain-0.1.0.exe
  • For Mac OSX, take all of the files from the following zip archive: FreeEMS-ToolChain-MacOSX-Fink.zip and place them in this directory: /sw/fink/dists/local/main/finkinfo/ then run fink install ...similar to debian...
  • Additionally, you can get the source from Sean Keys master branch (or any fork thereof) on github: XGATE Binutils
